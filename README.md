# plans-d-action
## MANIPULATION DES TABLEAUX :
### Partie 1 :
 [] : pour creer un tableau vide d'une maniere simple 
 new Array() : fonction predefinie pour creer un tableau
 .lenght : nombre d'elements dans le tableau 
### Partie 2 : 
push() : ajouter un element a la fin du tableau 
pop() : supprimer le dernier element du tableau
shift() : supprimer le premier element du tableau 
unshift() : ajouter un element au debut du tableau
